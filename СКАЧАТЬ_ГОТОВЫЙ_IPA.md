# 📥 КАК СКАЧАТЬ ГОТОВЫЙ IPA (собран в облаке)

## ✅ Что было сделано

Я исправил GitHub Actions workflow для **автоматической сборки IPA в облаке на macOS**.

Теперь ты можешь:
1. **Запустить сборку** через GitHub Actions (на облачной машина GitHub)
2. **Скачать готовый IPA файл** на свой Windows
3. **Установить на iPhone** через iMazing прямо с Windows

**Не нужен Mac!** 🎉

---

## 🚀 БЫСТРЫЙ СПОСОБ: Запустить сборку через GitHub Actions

### Шаг 1: Откройте GitHub Actions

Перейди сюда:
```
https://github.com/ideoma-last/WordClockWidgets/actions
```

Или кликни на вкладке **Actions** в репозитории

### Шаг 2: Выбери workflow "Build iOS App"

1. В левой панели найди: **"Build iOS App"**
2. Кликни на неё

### Шаг 3: Запусти сборку

1. Кликни кнопку: **"Run workflow"** (серая кнопка сверху)
2. Выбери ветку: **ios-port** (должна быть по умолчанию)
3. Кликни: **"Run workflow"** (зелёная кнопка)

**Готово!** Сборка началась в облаке

---

## ⏱️ Сколько времени займет?

**5-10 минут** в зависимости от нагрузки GitHub Actions

Посмотри прогресс сборки:
- Жёлтый круг 🟡 = собирается
- Зелёная галочка ✅ = готово
- Красный крест ❌ = ошибка

---

## 📥 Шаг 4: Скачай готовый IPA

Когда сборка завершится (покажется ✅):

1. Кликни на job "Build iOS App - Generate .ipa for iPhone"
2. Прокрути вниз до секции **"Artifacts"**
3. Найди: **"WordClockWidgets-IPA"**
4. Кликни для скачивания

Тебе загрузится ZIP файл с:
- ✅ **WordClockWidgets.ipa** - готовый к установке IPA
- ✅ **README_INSTALL.md** - инструкции по установке

---

## 📱 Шаг 5: Установи на iPhone через iMazing (Windows)

### Требуется:
- Windows PC
- iPhone XS
- USB кабель или WiFi соединение
- [iMazing](https://imazing.com/) (скачать бесплатно)

### Инструкция:

1. **Скачай iMazing**
   - Перейди на https://imazing.com/
   - Download для Windows

2. **Установи iMazing**
   - Запусти установщик
   - Следуй инструкциям

3. **Подключи iPhone**
   - Подключи USB кабелем к Windows
   - Или используй WiFi соединение (если в одной сети)
   - На iPhone нажми "Trust" если спросит

4. **Откройн iMazing**
   - На главной странице найди твой iPhone
   - Кликни на него

5. **Установи IPA**
   - В левом меню найди: "Apps"
   - Кликни кнопку: "Install app" или "+"
   - Выбери загруженный файл **WordClockWidgets.ipa**
   - Жди завершения

6. **Дай разрешение на iPhone**
   - После установки откройi iPhone
   - Перейди в: **Settings** (⚙️ Параметры)
   - **General** (Основные)
   - **VPN & Device Management** (управление VPN и Устройствами)
   - Найди свой Apple ID
   - Нажми: **Trust** (Доверять)

7. **Готово!** 🎉
   - На home screen появится иконка приложения
   - Можешь открыть и использовать

## ❓ Часто задаваемые вопросы

### Q: Сколько времени займёт вся процедура?
**A:** 
- Сборка в облаке: 5-10 минут
- Скачивание IPA: 1-2 минуты (зависит от интернета)
- Установка на iPhone: 2-5 минут

**Итого: ~15 минут максимум**

### Q: Нужна ли мне Apple Developer Account?
**A:** Нет! Ты устанавливаешь через iMazing без разработчика. Нужен только Apple ID на iPhone.

### Q: Нужен ли Mac?
**A:** Нет! Сборка происходит на облачной машине GitHub (macOS). Тебе нужен Windows + iMazing.

### Q: Что если сборка упала?
**A:** Кликни "Run workflow" снова. Проверь логи ошибки (кликни на job). Обычно помогает повторная попытка.

### Q: Какими браузерами это работает?
**A:** Любой браузер на Windows. Chrome, Edge, Firefox - все работают одинаково.

### Q: iPhone не видит приложение после установки?
**A:** Убедись что ты нажал **Trust** в Device Management. Перезагрузи iPhone (выключи/включи).

### Q: Приложение установилось но сразу закрылось?
**A:** Обычно это значит что ты не нажал Trust. Проверь в Device Management и нажми Trust.

### Q: Можно ли переустановить приложение?
**A:** Да, столько раз сколько нужно. Просто повтори шаги 5-7.

## 🔍 ЕСЛИ ЧТО-ТО НЕ СРАБОТАЛО

### Сборка упала (красный крест ❌)

1. Кликни на workflow run
2. Посмотри логи (внизу страницы)
3. Найди "❌ Build failed"
4. Прочитай текст ошибки
5. Создай Issue с текстом ошибки на GitHub

### iMazing не видит iPhone

1. Переподключи USB кабель
2. На iPhone нажми "Trust"
3. Перезагрузи iPhone
4. Закройи iMazing и откройи снова
5. Проверь что используешь оригинальный USB кабель

### Приложение не открывается

1. Settings → General → VPN & Device Management
2. Нажми Trust на своём Apple ID
3. Перезагрузи iPhone
4. Сотри приложение и переустанови

### Не могу скачать IPA из GitHub

1. Убедись что сборка завершена (зелёная галочка ✅)
2. Обнови страницу в браузере (F5)
3. Кликни на "Run details" внизу workflow
4. Прокрути вниз до "Artifacts"
