# WordClockWidgets iOS

Это iOS-версия виджета WordClockWidgets, портированная с Android.

## Требования
- iOS 18.4.1 или выше
- Устройство: iPhone XS или совместимое
- Xcode 15+ (для сборки и установки)

## Установка
1. Откройте проект в Xcode на macOS.
2. Выберите устройство iPhone XS (или симулятор).
3. Соберите и запустите приложение.
4. Добавьте виджет на домашний экран через виджет-галерею.

## Функции
- Отображение времени словами на русском языке.
- Виджет обновляется автоматически.
- Конфигурационное приложение для настройки позиций элементов (в разработке).

## Совместимость
- Протестировано на iOS 18.4.1.
- Поддерживает виджеты WidgetKit.

## Бэкапы
Android-версия сохранена в ветке main и теге android-backup.
